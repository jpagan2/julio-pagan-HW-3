Steps to build program:

1. open the 445hw3 directory
2. execute 'sudo apt-get -y update'
3. execute 'sudo apt-get -y install openjdk-17-jdk'
4. execute 'sudo snap install gradle --classic'
5. execute './gradlew clean build'
6. execute './gradlew run'

- The script: build.sh performs steps 1-6
